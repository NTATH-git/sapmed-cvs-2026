# 2026_Homepage — 札医大 心臓血管外科 HP管理

## このディレクトリの目的
札幌医科大学 心臓血管外科 **教室ホームページ** の更新作業領域。
2026年の **新教授赴任** に合わせた更新が主目的。同門会（doumonkai）関連ページも管理する。

**このプロジェクトだけ git で管理されている**（他はGoogle Drive直）。

## 作業者プロフィール
- 札幌医科大学 心臓血管外科 中嶋智宏
- HP更新の運用担当

## ディレクトリの中身（要点）
| ファイル/フォルダ | 用途 |
|---------|-----|
| `.git/` | **git管理されている**（他のプロジェクトと違う点） |
| `2022年12月12日教室説明会.pptx` | 教室説明会資料（167MB） |
| `2026年/` | 2026年の集合写真・行事写真など（624エントリ） |

| `Photos-3-001/` | 個別写真フォルダ |

| `doumonkai-members.html`, `doumonkai-register.html`, `doumonkai-register-success.html` | 同門会員登録ページ |

## ファイル命名規則
- 写真: 日本語日付フォルダ `YYYY年MM月DD日（曜）...`
- HTML: 機能名のkebab-case（`doumonkai-members.html`）
- レポート: 年度別

## やってほしいこと
1. **教授交代に伴う文言・写真の更新**
2. **doumonkai HTML の編集**（同門会員管理画面）
3. **Annual Report の整形**
4. **写真フォルダの整理**（重複・不要写真の検出）

## 注意（重要）
- **git管理されている**ので、編集前に `git status` で状況確認、コミットは適切に
- **大容量ファイル**（.pptx 167MB、写真フォルダ数百枚）がある → `.gitignore` 確認、誤コミットしない
- **同門会員の個人情報**（氏名、連絡先）が HTML に含まれる可能性 → 公開前に必ずレビュー
- 写真は **顔・氏名** が映る → 公開設定の確認
